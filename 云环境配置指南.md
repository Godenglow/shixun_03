# 云环境配置指南

## 1. 获取云环境ID

### 方法1：通过微信开发者工具
1. 打开微信开发者工具
2. 点击右上角的 **"云开发"** 按钮
3. 进入云开发控制台
4. 在环境设置中查看 **"环境ID"**（格式类似：`xxxxx-xxxxx`）

### 方法2：控制台查看
如果已经启用云开发，可以在控制台中看到环境信息。

## 2. 更新配置

在 `miniprogram/app.js` 中找到这一行：
```javascript
env: 'test-1', // 替换为你的云环境ID
```

将 `'test-1'` 替换为您的实际云环境ID，例如：
```javascript
env: 'your-actual-env-id', // 替换为你的云环境ID
```

## 3. 如果还没有云环境

### 启用云开发
1. 在微信开发者工具中点击 **"云开发"** 按钮
2. 选择 **"开通云开发"**
3. 按照提示创建云环境
4. 等待云环境创建完成

### 环境命名
- 环境名称：可以自定义，如"心理测试小程序"
- 环境ID：系统自动生成，记录下来

## 4. 常见问题

### Q: 环境创建失败？
A: 确保微信开发者工具版本最新，并且有有效的微信小程序AppID

### Q: 云函数调用失败？
A: 
1. 检查环境ID是否正确
2. 确保云函数已部署
3. 检查网络连接

### Q: 数据库连接失败？
A: 
1. 确认云环境已启用
2. 环境ID配置正确
3. 检查云开发权限设置
